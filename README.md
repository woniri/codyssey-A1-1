# 🚀 나만의 프롬프트 관리 프로그램 (Prompt Manager)

GenAI 미션을 수행하며 파편화된 수많은 프롬프트들을 체계적으로 분류하고 관리하는 Python 콘솔 기반 프로그램입니다.

## 🛠️ 개발 환경
- **Language:** Python 3.10+
- **Editor:** VSCode
- **Version Control:** Git / GitHub

## ✨ 주요 기능
1. **프롬프트 추가:** 제목, 내용, 카테고리를 입력받아 새로운 프롬프트 등록 (빈 값 검증)
2. **프롬프트 목록 보기:** 등록된 모든 프롬프트의 카테고리, 제목, 즐겨찾기(⭐) 여부 출력
3. **카테고리별 조회:** 지정된 6개 카테고리별로 프롬프트를 필터링하여 출력
4. **프롬프트 검색:** 키워드를 기반으로 제목 및 내용에서 일치하는 프롬프트 검색
5. **프롬프트 상세 보기:** 번호 선택 시 프롬프트의 전체 내용 및 즐겨찾기 상태 출력
6. **즐겨찾기 관리:** 자주 사용하는 프롬프트를 즐겨찾기에 추가/해제 (토글 방식)
7. **즐겨찾기 목록 보기:** 즐겨찾기 등록된 항목들만 모아서 간결하게 출력
0. **종료

- 프로그램 실행 중 추가한 프롬프트와 즐겨찾기 상태는 종료 시 초기화됩니다.

## 📂 등록된 프롬프트 카테고리
- 텍스트 생성 / 이미지 생성 / 영상 생성 / 페르소나 / 자동화 / 기타
